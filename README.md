# Weather13

A Weather application with React, StyledComponents, Vite, Typescript and Vitest :)


Instructions:
git clone https://github.com/rafabrunodev/weather13.git
cd weather13

To Run the project:
yarn
yarn dev

To Tests:
yarn test
